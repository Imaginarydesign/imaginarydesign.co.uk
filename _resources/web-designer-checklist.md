## Requirements

- npm
- Gulp
- Bower

## Installation

```sh
$ git clone https://github.com/Imaginarydesign/webdev-checklist.git
$ cd web-check
$ bower install
$ npm install
```
To run locally:

```sh
$ gulp serve
```
